# QR Code Generator ✅  A simple and responsive **QR Code Generator** built using **HTML, CSS, and JavaScript**.   Users can enter any text or URL, generate a QR code, and download it as an image.  ## ✨ Features ✅ Generate QR code from any text or URL   ✅ Download QR code as an image   ✅ Responsive UI
